# Email Assitant

## Motivation
Email overload is a serious problem in modern life. Users receive a large number of emails including critical work notifications, personal communications, as well as those news subscriptions, marketing advertisements. Categorizing these emails manually is time-consuming and labor-intensive. Whatâ€™s more, it is easy to miss important information. Therefore, users need an intelligent email helper to assist them with category and key words extraction.
## Methodology
### Workflow Agent

1. Extracted key information in the email, such as date, location, and contact information
2. Write potential response

### Prompt Engineering
